![Film](https://raw.githubusercontent.com/pierremtb/ionic-swapi/master/screenshots/film.png)

## Setting up
```console
$ git clone https://github.com/pierremtb/ionic-swapi
$ cd ionic-swapi
$ npm install
```

## Running (web)
```console
$ ionic serve
```

## Running (iOS)
```console
$ npm build --prod
$ ionic cap sync ios
$ ionic cap open ios
** Run on simulator or device in XCode **
```
